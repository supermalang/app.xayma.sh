# /n8n-workflow — Scaffold n8n Workflow

## Usage
```\n/n8n-workflow 5.3\n```\n\n## Purpose\nScaffold a new n8n workflow with input/output contract, error handling, and Kafka integration.\n\n## Output\nGenerates workflow JSON with:\n- Input schema (validated webhook request)\n- Output schema (typed response)\n- Error handling paths\n- Kafka producer (if credit operation)\n- Logging and retry logic\n\n## Workflow Template\n\n```json\n{\n  \"name\": \"Credit Topup Webhook\",\n  \"active\": false,\n  \"nodes\": [\n    {\n      \"name\": \"Webhook In\",\n      \"type\": \"n8n-nodes-base.webhook\",\n      \"typeVersion\": 2,\n      \"position\": [250, 300],\n      \"parameters\": {\n        \"path\": \"credit-topup\",\n        \"httpMethod\": \"POST\",\n        \"authentication\": \"genericCredentialType\",\n        \"genericCredentialType\": \"httpHeaderAuth\"\n      }\n    },\n    {\n      \"name\": \"Validate Input\",\n      \"type\": \"n8n-nodes-base.if\",\n      \"typeVersion\": 2,\n      \"position\": [450, 300],\n      \"parameters\": {\n        \"conditions\": {\n          \"string\": [\n            {\n              \"value1\": \"{{ $json.partnerId }}\",\n              \"operation\": \"isEmpty\",\n              \"value2\": null\n            }\n          ]\n        }\n      }\n    },\n    {\n      \"name\": \"Error Response\",\n      \"type\": \"n8n-nodes-base.webhook\",\n      \"typeVersion\": 1,\n      \"position\": [650, 150],\n      \"parameters\": {\n        \"method\": \"POST\",\n        \"contentType\": \"application/json\"\n      },\n      \"incoming\": [\"If - Idempotency Check\n```javascript\n// Before creating transaction\nconst existing = await supabase\n  .from('xayma_app.transactions')\n  .select('id, newBalance')\n  .eq('paymentRef', paymentRef)\n  .single()\n\nif (existing.data) {\n  // Return cached result\n  return { success: true, transactionId: existing.data.id, ... }\n}\n\n// Create new transaction\nconst { data } = await supabase\n  .from('xayma_app.transactions')\n  .insert([...]).select().single()\n```\n\n### 2. Kafka Publishing\n```javascript\n// Publish event for consumer\nawait kafka.producer.send({\n  topic: 'credit_events',\n  messages: [\n    {\n      key: partnerId,\n      value: JSON.stringify({\n        type: 'credit_topup',\n        partnerId,\n        amount,\n        timestamp: new Date().toISOString()\n      })\n    }\n  ]\n})\n```\n\n### 3. Error Response\n```javascript\n// Localized error for frontend\nreturn {\n  success: false,\n  error: {\n    code: 'INSUFFICIENT_FUNDS',\n    message: 'errors.credit_insufficient'  // Vue i18n key\n  }\n}\n```\n\n## Validation Checklist\n- [ ] Input schema validates request shape\n- [ ] All fields required/optional documented\n- [ ] Output schema matches contract\n- [ ] Error cases handled (validation, DB, Kafka)\n- [ ] Idempotency implemented (paymentRef dedup)\n- [ ] Kafka event published for credit updates\n- [ ] Sensitive data removed from logs\n- [ ] Timeouts configured on HTTP requests\n- [ ] Service role key used (n8n environment only)\n- [ ] Workflow tested end-to-end\n\n## Testing the Workflow\n\n```bash\n# Via n8n UI\n1.
